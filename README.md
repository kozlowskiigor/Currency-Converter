## **Description**

In **[Simple currency converter](https://kozlowskiigor.github.io/Currency-Converter/currencyConverter.html)** you can convert different currencies. 
There are currently 5 currencies available:
- PLN
- EUR
- GBP 
- CZK
- USD.
